# Assignment 5 — Clustering and Initialization

This repository contains the implementation and report for Assignment 5 of the Statistical Methods for Machine Learning course.

## Project Objective

The project studies the effect of initialization on clustering performance by comparing standard k-means and k-means++ across multiple experiments.

## Contents

- `KMEANS-Zeinab.ipynb` : Complete notebook implementation
- `Zeinab_Kmeans.pdf` : Final report
- Figures used throughout the experiments

## Implemented Methods

- Standard k-means
- k-means++
- Vectorized distance computation

## Experiments

The notebook includes:
- Multiple clustering trials
- Inertia comparison
- Stability analysis
- Convergence analysis
- Overlapping clusters
- High-dimensional datasets
